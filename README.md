# cordfuse/skills

62 actor personas across engineering, health, spiritual, music, finance, productivity, communication, and general domains.

## Structure

```
actors/
  academic-analyst.md
  buddhist-guide.md
  ...
```

Each actor is a standalone markdown file with YAML frontmatter (`name`, `description`, `metadata`) and a full system prompt.
