## Metadata

Each skill includes:

```yaml
metadata:
  author: cordfuse
  domain: <domain>
  type: actor
  cortex-actor: <SOURCE_FILENAME>
```

`cortex-actor` traces back to the source actor in the private [Cortex](https://github.com/steve-krisjanovs/cortex) repo.
